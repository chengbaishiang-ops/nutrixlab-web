# 🧪 NutriXLab

AI-powered nutrition analysis tool  
Scan supplement labels. Get instant, science-based insights.

---

## 🚀 Live Demo

## ✨ Features

- 📸 OCR ingredient recognition  
- 🧠 AI-based analysis engine  
- 📊 Scientific scoring system  
- ⚠️ Safety evaluation  
- ⚡ Instant results in browser  

---

## 🧩 How It Works

1. Upload supplement label  
2. OCR extracts ingredients  
3. AI analyzes composition  
4. Generate safety & effectiveness score
